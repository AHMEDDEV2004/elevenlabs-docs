curl -X 'GET' \

```Python Python
import requests

url = "https://api.elevenlabs.io/v1/user"

headers = {
  "Accept": "application/json",
  "xi-api-key": "<xi-api-key>"
}

response = requests.get(url, headers=headers)

print(response.text)
```
