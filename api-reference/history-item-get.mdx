<RequestExample>

```Bash Curl
curl -X 'GET' \
  'https://api.elevenlabs.io/v1/history/<history-item-id>' \
  --header 'accept: application/json' \
  --header 'xi-api-key: <xi-api-key>'
```

```Python Python
import requests

url = "https://api.elevenlabs.io/v1/history/<history-item-id>"

headers = {
  "Accept": "application/json",
  "xi-api-key": "<xi-api-key>"
}

response = requests.get(url, headers=headers)

print(response.text)
```
