<RequestExample>

```Bash Curl
curl -X 'POST' \
  'https://api.elevenlabs.io/v1/history/download' \
  --header 'accept: */*' \
  --header 'xi-api-key: <xi-api-key>' \
  --header 'Content-Type: application/json' \
  --data '{
    "history_item_ids": [
      "<history-item-id>"
    ]
  }'
```

```Python Python
import requests

url = "https://api.elevenlabs.io/v1/history/download"

headers = {
  "Accept": "*/*",
  "Content-Type": "application/json",
  "xi-api-key": "<xi-api-key>"
}

data = {
  "history_item_ids": [
    "<history-item-id>"
  ]
}

response = requests.post(url, json=data, headers=headers)

print(response.text)
```
