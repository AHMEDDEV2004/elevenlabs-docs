<RequestExample>

```Bash Curl
curl -X 'DELETE' \
  'https://api.elevenlabs.io/v1/voices/<voice-id>' \
  --header 'accept: application/json' \
  --header 'xi-api-key: <xi-api-key>'
```

```Python Python
import requests

url = "https://api.elevenlabs.io/v1/voices/<voice-id>"

headers = {
  "Accept": "application/json",
  "xi-api-key": "<xi-api-key>"
}

response = requests.delete(url, headers=headers)

print(response.text)
```

```Javascript javascript
async function Delete_Voice() {
    return fetch('https://api.elevenlabs.io/v1/voices/<voice-id>', {
        method: 'DELETE',
        headers: {
            'accept': 'application/json',
            'xi-api-key': '<xi-api-key>'
        }
    }).then(response => response.text()).then(text => JSON.parse(text))
}

response = await Delete_Voice()
console.log(response)
```
